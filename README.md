# 2048

A simple and polished web implementation of the classic 2048 puzzle game. The project combines HTML, SCSS, and JavaScript to deliver an interactive experience where players merge matching tiles, track their score, and try to reach the 2048 tile.

## Live Preview

- [Open the game](https://your-username.github.io/FrontEnd_Projects_NB/)

## Technologies Used

- HTML5
- CSS3 / SCSS
- JavaScript (ES6+)
- Parcel

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/FrontEnd_Projects_NB.git
cd FrontEnd_Projects_NB
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

## Features

- Responsive game board with smooth tile movement
- Keyboard controls for arrow keys
- Score tracking and win/lose messages
- Restart functionality for quick playthroughs
